# Structure Evaluation Rubric

Evaluate if the research output follows proper markdown heading structure.

## Evaluation Priority
1. **If file_content is not null/empty**: Evaluate the file_content primarily
2. **If file_content is null/empty**: Evaluate the research_summary

## Criterion
Does the output follow proper markdown heading structure (## for main topics, ### for sub-topics)?

## Scoring
- **1.0**: Proper heading hierarchy with ## and ### used correctly
- **0.5**: Has headings but inconsistent or improper hierarchy
- **0.0**: No headings or incorrect markdown heading format
